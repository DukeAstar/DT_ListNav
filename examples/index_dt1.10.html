DT_Listnav example

Preamble

This example shows the basic use case of the DT_Listnav plug-in. With DT_Listnav enabled for a table, the user has the ability to click on a letter to filter on the choice he made.
By default the filter is on the second column.

		"sDom": 'Zlfrtip',
		"bServerSide" : true,

		}
